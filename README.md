# EV Stock Price Comparison (2021–2024)
**Python • Pandas • Matplotlib • Financial Data Analysis**

## 📌 Overview
This project analyzes the stock price performance of major electric vehicle companies — **Tesla (TSLA), Rivian (RIVN), and Lucid (LCID)** — from 2021 to 2024.  
The goal is to compare volatility, returns, and long‑term trends to understand how each company performed during a period of rapid EV market growth and economic uncertainty.

This project demonstrates skills in **time‑series analysis, financial data handling, visualization, and market interpretation**.

---

## 📊 Dataset
Stock data was collected using the **yfinance** Python library, including:
- Daily closing prices  
- Volume  
- High/low prices  
- Adjusted close  

Time period: **January 2021 – January 2024**

---

## 🔍 Analysis Steps
1. Imported stock data using `yfinance`  
2. Cleaned and structured the dataset  
3. Calculated:
   - Daily returns  
   - Cumulative returns  
   - Moving averages  
   - Volatility  
4. Visualized:
   - Price trends  
   - Return comparisons  
   - Moving average crossovers  
5. Interpreted market behavior and performance differences  

---

---

## 📌 Key Insights
- **Tesla** showed the strongest long‑term performance but also the highest volatility.  
- **Rivian** experienced significant early hype followed by sharp corrections.  
- **Lucid** showed weaker long‑term momentum and higher downside risk.  
- Market trends were heavily influenced by:
  - Interest rate hikes  
  - EV market competition  
  - Supply chain issues  
  - Investor sentiment shifts
